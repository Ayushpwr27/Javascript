# Unit Conversion 

This is a basic Javascript code that performs some basic calculations to convert Length, Volume ,and Mass

## conversion 

#### Length 
    1 meters = 3.281 feet 
    1 feet = 0.305 meters

#### Volume 
    1 liters = 0.264 gallon 
    1 gallon = 3.785 liters

#### Mass
    1 kilos = 2.205 pound
    1 pounds = 0.454 kilos
